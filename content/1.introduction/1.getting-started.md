# Getting Started

From your terminal to a deployed website in few minutes.

## Installation

::code-group

  ```bash [pnpm]
  # Using pnpm
  pnpm add -g create-ao-app
  ```
  ```bash [npm]
  # Using npm
  npm install -g create-ao-app
  ```
  ```bash [yarn]
  # Using yarn
  yarn add -g create-ao-app
  ```
  ```bash [npx]
  # Or use directly with npx
  npx create-ao-app init my-app
  ```
::
