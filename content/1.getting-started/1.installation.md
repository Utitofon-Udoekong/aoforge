# Getting Started

From your terminal to a deployed AO application in few minutes.

## Installation

::code-group

  ```bash [pnpm]
  # Using pnpm
  pnpm add -g ao-forge
  ```
  ```bash [npm]
  # Using npm
  npm install -g ao-forge
  ```
  ```bash [yarn]
  # Using yarn
  yarn add -g ao-forge
  ```
  ```bash [npx]
  # Or use directly with npx
  npx ao-forge init my-app
  ```
::
